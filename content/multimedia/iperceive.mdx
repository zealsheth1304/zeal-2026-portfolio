# Description

<div className="text-ds-b1 my-8"> 
    In this project, we created an environment for users to explore perception through various levels of immersion. 
    For this we re-create the traditional iSpy puzzle game for ‘Spooky Horror House‘ theme across different media from book to projection 
    mapping to VR experience. The objective of the exhibit design was to help users experience ‘Perception as a result of Actionist Theory’.
</div>

# My Learnings

We learnt a lot about Projection Mapping design technicalities like light interference, occlusion, resolution, detection. We also learnt the importance of user testing to understand the user flow through unexpected paths and designing a robust system.
