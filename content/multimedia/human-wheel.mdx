# Description

<div className="text-ds-b1 my-8"> 
    The goal of this project was to build an interactive installation that is fun & engaging. We decided to create an interactive parallel world of hamsters - Human Wheel. The idea is that 'Humans act as the Reverse Hamsters’ and interact with the Hamster House interface to trigger different hamster actions. Thus, creating their own story. This piece is inspired from Neil Mendoza’s Disruptive Devices.
</div>

# My Learnings

We learnt how to use the storytelling arc narrative to keep the user engaged and hooked. Learnt designing for a narrative. For eg. to add the SURPRISE factor by randomizing the hamster behavior on user inputs, to add the WOW factor, such that not only the users control ‘random’ actions in the hamster house, but also the hamster gives cues to user to perform different actions, to increase the comic time and relatability by adding real-life responses.
