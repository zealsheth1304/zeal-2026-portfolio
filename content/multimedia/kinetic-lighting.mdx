# Description

<div className="text-ds-b1 my-8"> 
Kinetic Lighting is an interactive installation that lights up using user’s kinetic energy. For this project, we decided to take inspiration from the ferris wheel and create a playful element that creates energy in the form of electricity and light with rotation of the wheels. We decided to use a large bicycle wheel to reduce the muscle effort, improve the smoothness and create more power. When the wheel rotates, the muscle power from user gets transferred into the stepper motor that acts as the generator that is used for lightings to create optical illusion, psychedelic patterns using mirrors and persistence of motion.
</div>
