# Description

<div className="text-ds-b1 my-8"> 
 This project, provides user an opportunity to engage their tactile, auditory and visual senses giving them fun and delightful experience. For tactile, sand is used as a stress buster for user. User are encouraged to create art work and deform the sand. User hand movement will be tracked by using kinect depth sensor and contour tracking using OpenFrameworks and input will be used to change the sound and visuals. Thus users can create interesting visuals and produce interesting sounds to make with their hand movements. Visuals will be inspired from particle effect and perlin noise. These visuals are projected on the sand.
</div>
