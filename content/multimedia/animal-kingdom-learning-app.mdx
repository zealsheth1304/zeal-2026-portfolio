# Description

<div className="text-ds-b1 my-8"> 
    For this project, we designed an interactive app for kids(4-6yrs) to make learning the language fun and engaging. It has been found, scientifically, that teaching kids(4-6yrs) about animal names and sounds helps them develop their grasp over language. To make it a fun experience for the kids our app uses voice assistance, bright colors, motion graphics, fun visuals and positive encouraging language.
