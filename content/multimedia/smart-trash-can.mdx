# Description

<div className="text-ds-b1 my-8"> 
The world produces over 2 billion tonnes of solid waste every year. This produces greenhouse gases causing global warming. The goal of the project was to design a system to reduce this solid waste produced & raise climate awareness.
    We designed and implemented an automatic trash type detector system using machine learning. The system uses a camera to capture images of trash and classifies them into different categories such as recyclable, compostable, and landfill waste. The system is designed to be user-friendly and can be easily integrated into existing waste management systems.
</div>
